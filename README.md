# Earthquakes 1900–2013 Dashboard

## Overview
This project presents an interactive **Tableau dashboard** analyzing global earthquake data from **1900 to 2013**.  
It visualizes seismic activity trends across magnitude, depth, time, and location — helping identify patterns and correlations in earthquake behavior worldwide.

## Objectives
- **Visualize earthquake frequency** — total events, magnitude distribution, and depth variation.  
- **Analyze temporal trends** — earthquakes by year and month to reveal seasonal or long‑term patterns.  
- **Explore spatial distribution** — global map showing earthquake hotspots.  
- **Understand relationships** — between depth and magnitude, and between event types over time.

## Key Insights
| Metric | Value | Description |
|--------|-------|-------------|
| **Total Earthquakes** | 7,480 | Recorded events between 1927–2014 |
| **Average Depth** | 78.01 km | Typical depth of seismic activity |
| **Average Magnitude Range** | 6.0–9.5 | Moderate to major earthquakes |
| **Average NST** | 272.6 | Average number of seismic stations reporting |
| **Max Gap** | 230.4 | Largest data gap between events |
| **Average RMS** | 1.008 | Mean root‑mean‑square error for readings |

## Dashboard Sections
- **Magnitude Histogram** — Displays earthquake counts by magnitude (6.0–9.5).  
- **Depth in KM Histogram** — Shows frequency of earthquakes at various depths.  
- **Relation Between Depth and Magnitude** — Scatter plot illustrating correlation between depth and intensity.  
- **Total Earthquakes by Year and Type** — Area chart showing seismic trends over time.  
- **Total Earthquakes by Month and Type** — Stacked bar chart revealing monthly patterns.  
- **Total Earthquakes by Place** — Interactive world map highlighting earthquake locations.

## Tools & Technologies
- **Tableau Desktop** — Data visualization and dashboard design  
- **CSV Dataset** — Earthquake records (1900–2013)  
- **GitHub** — Portfolio hosting and documentation  

## How to View
1. Download the `.twbx` file and open it in Tableau Desktop.  
2. Or view the interactive version on **Tableau Public** (link to be added).  
3. Use filters (date range, magnitude, depth) to explore detailed insights.
